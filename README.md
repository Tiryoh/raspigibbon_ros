# raspigibbon_ros

## About

Raspberry Pi Gibbon Controller + Raspberry Pi Gibbon Simulator suites runs on Gazebo

## Requirements

requires the following to run:

* Ubuntu
  * Ubuntu Trusty 14.04
* ROS
  * ROS Indigo
* Gazebo
  * Gazebo 2.x
