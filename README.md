# raspigibbon_ros

## About

Raspberry Pi Gibbon Controller suites runs on Gazebo

## Requirements

requires the following to run controller on Raspberry Pi 3:

* Ubuntu
  * Ubuntu Xenial 16.04
    * Ubuntu MATE 16.04.1 recomended
* ROS
  * ROS Kinetic
